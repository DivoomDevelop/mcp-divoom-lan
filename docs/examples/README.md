# Examples Bundle

This folder contains a curated subset of request/response examples for `mcp-divoom-lan`.

## Structure

- `api_catalog.json`: machine-readable example index
- `requests/divoom_api`: common JSON command requests
- `requests/multipart`: multipart first-part JSON examples
- `responses`: representative response shapes

## Notes

- These response files are illustrative shapes, not guaranteed dumps from your exact firmware.
- Always verify behavior on your device with live calls.
- For write operations, follow read-before-write workflow.
