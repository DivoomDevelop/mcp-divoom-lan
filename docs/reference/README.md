- safe operation order
- risky operations and guard rails

Included files:

- `Divoom_Watchface_Remote_Customization_Guide.md` (ZH condensed)
- `Divoom_Watchface_Remote_Customization_Guide_EN.md` (EN condensed)
- `guide-key-points.zh.md`
- `guide-key-points.en.md`

For full protocol context, keep the original upstream guide as the primary source of truth in your source repository.
